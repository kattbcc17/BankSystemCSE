Overview
--------
This project is a Bank Management System implemented in C++. It allows users to manage bank accounts, including adding, updating, and removing accounts, as well as performing transactions like deposits and withdrawals. The system also supports features for finding accounts, calculating total balances, and applying dividends.

Features
--------
1. Add Item: Add a new item to the inventory.
2. Update Item: Update the details of an existing item.
3. Remove Item: Remove an item from the inventory.
4. Update Quantity: Update the quantity of an existing item.
5. Generate Report: Generate a report of all items in the inventory.
6. Search Item: Search for an item by name.
7. Filter Items: Filter items based on their description.
8. Save Inventory: Save the current inventory to a file.
9. Load Inventory: Load inventory data from a file.
10. Search for accounts by balance range.
11. Transfer funds between accounts.

Usage
-----
1. Compile the project using a C++ compiler (e.g., g++).
Example: g++ -o BankSystem main.cpp Account.cpp Contact.cpp
2. Run the compiled program.
Example: ./BankSystem
3. Follow the on-screen prompts to manage bank accounts and perform transactions.

File Descriptions
-----------------
1.main.cpp: The main entry point of the program, containing the user interface and main loop.
2. Account.hpp: Header file defining the Account class and its methods.
3. Account.cpp: Implementation file containing the definitions for the Account class methods.
4. Contact.hpp: Header file defining the Contact class and its methods.
5. Contact.cpp: Implementation file containing the definitions for the Contact class methods.
